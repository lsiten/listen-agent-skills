## AI核心能力

本技能的核心是AI助手的能力，而不是脚本。AI助手通过以下方式完成任务：

- **独立思考** - 运用知识和推理能力，独立分析和解决问题
- **理解分析** - 理解任务需求，分析问题本质，识别关键信息
- **规划执行** - 制定执行计划，按步骤完成任务
- **使用工具** - 在需要时使用脚本、MCP工具等辅助资源

## 使用方法

### 与AI助手交互

直接向AI助手描述你的需求，AI会：

1. 理解你的需求
2. 分析任务要求
3. 独立完成主要工作
4. 在需要时使用辅助工具
5. 提供结果和反馈

### 辅助工具

脚本和工具是辅助性的，主要用于：

- 安装依赖（如需要）
- 格式化输出（如需要）
- 确定性操作（如需要）

**重要**：主要工作应该由AI助手通过思考和推理来完成。
